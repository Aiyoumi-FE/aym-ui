### props 说明


| 参数      |类型| 默认值    | 可选值|描述    | 
|:-------- | :--------|:--------  |:--------|:---------|    
| data |Array| []  |-| 数据 |  
| title |String| -  |-| 标题 |   
| confirmTxt |String| -  |-| 确定按钮文本 |  
| selectedIndex |Array| []|-| 选中索引 |  
| onSelect |Function| -|-| 确定事件回调 |  
| onCancel |Function| -|-| 取消事件回调 |  
| onChange |Function| -|-| 选中变化事件回调 |
