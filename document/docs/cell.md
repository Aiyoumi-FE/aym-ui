### props 说明

| 参数      |类型| 默认值    | 可选值|描述    | 
|:-------- | :--------|:--------  |:--------|:---------|    
| title |String| -  |-| 左侧描述文字 |   
| value |String| -  |-| 右侧值 |   
| isLink |Boolean| false|-| 是否显示右侧箭头 |   
| link |String,Object| -|-| 跳转地址 |   
| isSwitch |Boolean| -|-| 是否包含switch |   
| disabled |Boolean| false|-| 是否可跳转 |   
| isSelect |Boolean| false|-| 是否完整（填完或选中为黑色，默认灰色） |
